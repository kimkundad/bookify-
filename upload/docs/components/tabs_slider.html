Tabs Slider

Tabs slider

Small extention for bootstrap tabs plugin https://getbootstrap.com/docs/3.3/javascript/#tabs

You can find example on index layout 6.

First you need to add "slideTabs" id to tabs navigation

<ul class="nav nav-tabs" id="slideTabs" role="tablist">
    <li class="active" role="presentation">
        <a
            aria-controls="tab-1"
            role="tab"
            data-toggle="tab"
            href="#tab-1"
        ></a>
    </li>
    <li class="active" role="presentation">
        <a
            aria-controls="tab-2"
            role="tab"
            data-toggle="tab"
            href="#tab-2"
        ></a>
    </li>
</ul>

Then create a div (where you want to display your slider) with id of "slideTabsSlides" for slides with images and link images to nav items with data-tab attribute

<div id="slideTabsSlides">
    <img 
        class="active" 
        src="path/to/first/image" 
        alt="Image Alternative text" 
        title="Image Title" 
        data-tab="tab-1"/>
    <img 
        class="active" 
        src="path/to/second/image" 
        alt="Image Alternative text" 
        title="Image Title" 
        data-tab="tab-2"/>
</div>
