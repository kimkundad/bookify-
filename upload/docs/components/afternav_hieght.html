Afternav Height

After nav height

Small plugin to make any element height of widow minus navigation height. Handy for one screen layouts (for example map view layout)

Code is pretty simple just add "afternav-height" class to element

<div class="afternav-height">
    // div will be height of the window exept navigation
</div>
