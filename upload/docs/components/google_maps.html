Google Maps

Google maps

Based (you guess) on google maps API. You can find full documentation here https://developers.google.com/maps/documentation/

I have created small facade for using it with theme.

Basic usage 

<div class="google-map" data-lat="40.5646852" data-lng="-73.9954863">
</div>

"data-lat" and "data-lng" is used for specifing latitude and longitude of position you want to display
